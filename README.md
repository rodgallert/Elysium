# Elysium

**English** · [Português](./README.pt-BR.md)

> *In the World of Darkness, an Elysium is neutral, sacred ground where the Kindred set aside their rivalries and coexist.*

Elysium is my collection of self-contained demonstration projects. Each one lives in its own folder,
runs independently, ships with its own stack and documentation, and takes its name from the World of
Darkness. Click any project below to open its own README.

## Projects

| _…future_ | _Coming soon_ | | |

> (e.g. *Boon* — a formal debt/favor traded between Kindred, fitting for a transactions platform).

## About

Portfolio projects demonstrating engineering practice end to end: architecture decisions (see each
project's `docs/adr/`), automated testing, observability, and pragmatic scope. Each project README
explains its own design, tradeoffs, and how to run it.
