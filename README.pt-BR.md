# Elysium

> *No World of Darkness, o Elysium é território neutro e sagrado, onde os Kindred deixam de lado suas rivalidades e coexistem.*

Elysium é minha coleção de projetos demonstrativos autocontidos. Cada um vive em sua própria pasta,
roda de forma independente, traz sua própria stack e documentação, e recebe um nome vindo do World
of Darkness. Clique em qualquer projeto abaixo para abrir o README dele.

## Projetos

| Projeto | O que é | Stack | Status |
|---------|---------|-------|--------|
| [**Boon**](./boon/README.pt-BR.md) | Plataforma de venda de conteúdo digital — gateway de borda, dois SPAs, workers assíncronos segregados, abstração de gateway de pagamento | Ruby on Rails · Vue 3 + Vuetify · Sidekiq/Redis · PostgreSQL · Nginx *(worker em Go planejado)* | 🚧 Em andamento |
| _…futuros_ | _Em breve_ | | |

> **Nomenclatura:** o guarda-chuva é o **Elysium**; cada projeto recebe um nome vindo da mitologia
> do World of Darkness (ex.: *Boon* — uma dívida/favor formal negociado entre os Kindred, apropriado
> para uma plataforma de transações).

## Sobre

Projetos de portfólio que demonstram prática de engenharia de ponta a ponta: decisões de arquitetura
(veja o `docs/adr/` de cada projeto), testes automatizados, observabilidade e escopo pragmático. O
README de cada projeto explica seu próprio design, tradeoffs e como executá-lo.
